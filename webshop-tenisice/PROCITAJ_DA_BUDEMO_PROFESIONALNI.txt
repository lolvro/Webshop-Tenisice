# webshop-tenisice



## backend

1. otvori terminal (VS code ili sta vec koristis)u folderu backend

2. instaliraj pakete: npm install

3. pokreni server: node server.js

-endpointi:

GET /tenisice

POST /tenisice

PUT /tenisice/:id

DELETE /tenisice/:id



## frontend

folder: frontend

otvori index.html u browseru ili preko Live Servera u VS

igraj se u styles.css
